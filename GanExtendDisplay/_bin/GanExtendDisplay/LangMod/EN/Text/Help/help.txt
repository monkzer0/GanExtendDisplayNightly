GanExtendDisplayNightly adds detailed stat, buff, and feat tooltips to character hover text, item inspections, and enchantment views. A community fork of GanExtendDisplay, maintained for the Elin Nightly build.

{topic|⚠ Compatibility}
Do not use this mod alongside the original GanExtendDisplay. Both patch the same game methods — running them together produces garbled or duplicated output. Only one should be active at a time.
Unsubscribe from the original before installing this one.

$What It Shows
{topic|Characters}
Hover over any character to see:
・ Level (color-coded threat; ☠ marks extreme danger)
・ Rarity badge (★ ☆ ◇ △)
・ Affinity (♥ / ♡)
・ Line 1: Sex, age, race, job, AI behavior, armor skill, attack style
・ Line 2: HP, DV (dodge), PV (protection), Speed
・ Line 3: SP (stamina), hunger, works and hobbies
・ Line 4: MP, carried weight vs. limit, EXP to next level
・ Resistances: all non-zero elemental resistances
・ Attributes: the eight core stats with values
・ Favorite gift: preferred category and food item
・ Conditions: active buffs and debuffs, color-coded green/red, with phase and numeric value — including area-effect debuffs the vanilla UI doesn't show
・ Acts: active combat abilities
・ Feats: passive traits and talents (own line, distinct color, toggled independently from acts)

{topic|Items on the Ground}
・ Level, rarity, and material

{topic|Inventory and Equipment}
・ Numeric values of all enchantment entries, including gene-type items

{topic|Buff Bar}
・ Status effects show extra detail on hover

$Controls
{pair|Hold Alt|Reveal lines set to Hide}
{pair|Alt + Caps Lock|Toggle "always show hidden lines" on/off}
{pair|Alt + End|Toggle the entire mod on/off}
{pair|Alt + Home|Toggle the game's debug display}

$Configuration
Config file location:
Elin\BepInEx\config\ExtendDisplay.cfg

If you install Mod Options (Steam Workshop ID 3381182341, by EvilMask), an "Extend Display" tab appears in its in-game panel. Mod Options is optional — the mod works without it.

{topic|Mod Options live-update behaviour}
{pair|Character line settings|Take effect immediately (mode, font size, faction filter, items per line)}
{pair|Main feature toggles|Take effect on the next game restart (controls whether patches load at startup)}

You can also edit settings live in the BepInEx config manager if you have it installed.

{topic|Display values}
{pair|Keep|Always visible}
{pair|Hide|Hidden by default; hold Alt to reveal}
{pair|Disable|Never shown}

{topic|Per-line settings (character lines)}
{pair|PCFactionOnly|When true, only shows for your faction members}
{pair|Size|Font size for that line}
{pair|Items Per Line|Acts / feats only: wrap to a new line after N entries (0 = no limit)}

Config descriptions appear in English, Simplified Chinese, and Japanese in the config manager.
